# NovaCS-Recruitment-Assesment-Application-
This project is a recruitment assessment platform which is developed for organisations evaluate candidates through structed, skills-based tasks. This repository includes a mapped sitemap for both recruits and candidates and all the front-end and back-end code along with notes, screenshots and information taken along the way.
